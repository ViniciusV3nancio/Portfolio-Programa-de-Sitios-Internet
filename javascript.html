O que é o Javascript?

O JavaScript é uma linguagem de programação de alto nível que foi criada inicialmente para ser executada em navegadores e manipular comportamentos de páginas web¹². É uma linguagem leve, interpretada, orientada a objetos, baseada em protótipos e em funções de primeira classe¹. Com o JavaScript, é possível incluir elementos dinâmicos em uma página estática, como mapas, formulários, operações numéricas, animações, infográficos interativos e muito mais¹. Ele é uma das tecnologias mais importantes voltadas para o front-end e, juntamente com HTML, CSS e PHP, forma um grupo de linguagens que abrangem praticamente todas as exigências do desenvolvimento de uma página completa, dinâmica e com boa performance¹. Além disso, o JavaScript não se limita apenas às páginas e aos navegadores. Com o advento de diversos frameworks, APIs, melhorias e criação de centenas de funções, hoje já é possível utilizar JavaScript em aplicativos mobile, softwares para desktop e até mesmo em back-end.

Math.Round

O método `Math.round(x)` em JavaScript é usado para arredondar um número para o inteiro mais próximo. Se a parte decimal do número é 0.5 ou superior, ele arredonda para cima para o próximo número inteiro. Se a parte decimal do número é menor que 0.5, ele arredonda para baixo para o número inteiro anterior.
Por exemplo:
- `Math.round(20.49)` retornará `20`
- `Math.round(20.5)` retornará `21`
- `Math.round(-20.5)` retornará `-20`
- `Math.round(-20.51)` retornará `-21`
Esse método é muito útil quando você precisa arredondar um número para o inteiro mais próximo em suas operações matemáticas.

GetElementByID

O getElementById é um método em JavaScript que faz parte da interface Document. Ele é usado para selecionar um elemento na página HTML com base no valor do atributo id desse elemento.


var elemento = document.getElementById('exemplo');

O document: É um objeto que representa o documento HTML carregado na janela. Ele fornece métodos para interagir com o conteúdo da página.
O getElementById('exemplo'): Este método procura e retorna uma referência para o primeiro elemento com o atributo id igual a "exemplo". Se nenhum elemento for encontrado, ele retorna null.

InnerHTML

Em JavaScript, o innerHTML é uma propriedade de objetos DOM (Document Object Model) que permite acessar ou modificar o conteúdo HTML de um elemento. Basicamente, o innerHTML fornece uma maneira de obter ou definir o HTML dentro de um elemento HTML.
Exemplo:

var elemento = document.getElementById('exemplo');
var conteudoHTML = elemento.innerHTML;
console.log(conteudoHTML);

While e DoWhile

A estrutura de controle de repetição while é utilizada em linguagens de programação para executar um bloco de código repetidamente enquanto uma condição específica for verdadeira


Exemplo

var contador = 0;
while (contador < 5) {
console.log("Número: " + contador);
contador++;
}


Uma variação do while é o do-while, que garante que o bloco de código seja executado pelo menos uma vez, mesmo se a condição for inicialmente falsa. A sintaxe é semelhante, mas a verificação da condição ocorre após a execução do bloco de código

Exemplo


var contador = 0;
do {
console.log("Número: " + contador);
contador++;
} while (contador < 5);

Switch Case

O switch é uma estrutura de controle de fluxo em muitas linguagens de programação, incluindo JavaScript. O switch é usado quando você tem várias condições e deseja executar diferentes blocos de código dependendo do valor de uma expressão.


Exemplo


var diaDaSemana = 3;
var mensagem;
switch (diaDaSemana) {
case 1:
mensagem = "Segunda-feira";
break;
case 2:
mensagem = "Terça-feira";
break;
case 3:
mensagem = "Quarta-feira";
break;
case 4:
mensagem = "Quinta-feira";
break;
case 5:
mensagem = "Sexta-feira";
break;
default:
mensagem = "Fim de semana";
}
console.log(mensagem);
